# tpmp-lab2-task3

## Overview
This project contains a C program that works with student data using structures.  
The program:
- stores information about 10 students (name, birth date, grades),
- sorts records by academic performance,
- determines which students are 20 years old or younger on a given date,
- prints results to the console.
